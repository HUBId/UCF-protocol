# Reason Code Catalog

This catalog documents the reason codes used by UCF records. Codes are
case-sensitive and should be treated as stable identifiers.

* `RC.GV.AAP.BLOCKED_BY_TRACE`
* `RC.GV.SHADOW.INVALID_LOOSENING`
